# АИС учета железнодорожных вагонов и составов

Минимально рабочая Django-версия для демонстрации на защите диплома. Приложение покрывает ключевой сценарий: вход по ролям, учет составов и вагонов, схема путей, перемещения с историей, загрузка документов, базовый OCR, отчеты CSV, уведомления и журнал операций.

## Быстрый запуск

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python manage.py migrate
python manage.py seed_demo
python manage.py runserver
```

Откройте `http://127.0.0.1:8000/login/`.

Демо-пользователи:

| Роль | Логин | Пароль |
| --- | --- | --- |
| Администратор | `admin` | `admin123` |
| Оператор | `operator` | `operator123` |
| Маневровый диспетчер | `dispatcher` | `dispatcher123` |

## PostgreSQL

По умолчанию для простого локального показа используется SQLite. Для соответствия ТЗ можно включить PostgreSQL переменными окружения:

```bash
export POSTGRES_DB=aisdip
export POSTGRES_USER=postgres
export POSTGRES_PASSWORD=postgres
export POSTGRES_HOST=localhost
export POSTGRES_PORT=5432
python manage.py migrate
```

## Демонстрационный сценарий

1. Войти под `admin/admin123` и открыть главную панель `/dashboard/`.
2. Показать справочники путей `/tracks/` и визуальную схему `/track-map/`.
3. Создать новый состав в `/trains/create/`.
4. Добавить вагон через `/wagons/create/`.
5. На `/track-map/` переместить состав или вагон на свободный участок.
6. Открыть карточку вагона и показать историю перемещений.
7. Загрузить документ в `/documents/upload/`, открыть OCR-страницу и запустить распознавание. Если Tesseract не установлен в ОС, система покажет управляемую ошибку и запишет уведомление.
8. Открыть `/reports/`, показать отчет и экспорт CSV.
9. Открыть `/admin-panel/` и показать журнал операций.

## OCR

Для настоящего распознавания должен быть установлен системный Tesseract OCR и языковые пакеты `rus`/`eng`. Даже при ошибке OCR документ сохраняется, статус меняется на `Ошибка`, а пользователь видит уведомление — этого достаточно для безопасной демонстрации обработки сбоя.
